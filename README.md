# 🥊 Бокс-Империя — 3D Boxing Game

Браузерная 3D-игра в бокс на Three.js. Ты против ИИ-бота на золотом ринге.

## 🎮 Играть
👉 **https://vmorkovka333-glitch.github.io/boxing-empire/**

Открывается прямо в браузере — на компьютере и телефоне, без установки.

## Управление
**Телефон:** белый круглый джойстик справа — движение; кнопки слева — джебы, хук, блок, уклон.

**Клавиатура:** `WASD` / стрелки — движение · `J` / `L` — джебы · `K` — хук · `Пробел` — блок · `Shift` — уклон.

## Как победить
- Первым нанести **50 чистых ударов**, либо
- **нокаут** (судья досчитает до 10), либо
- по очкам, если вышло время раунда (2:00).

Сделано на Three.js r134, один HTML-файл, без сборки.
